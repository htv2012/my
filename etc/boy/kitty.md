# Tab

- New: c+s+t, ⌘+t
- Close: c+s+q, ⌘+w
- Previous: c+s+left, s+⌘+[
- Next: c+s+right, s+⌘+]
- Next layout: c+s+L
- Title: c+s+a+t

# Window

- Resize: c+s+r
- Previous: c+s+[
